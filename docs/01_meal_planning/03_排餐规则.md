# 每周排餐规则

> 本规则为所有周计划的执行基准，新建周计划前务必核对。

## 规则一：主蛋白轮换

- 每顿主蛋白在 **猪肉 / 牛肉 / 鱼 / 虾 / 鹅肝** 之间交替，同日午晚尽量不重复。
- 每周配额：
  - 鱼：**2-3 次**
  - 虾：**2-3 次**
  - 鹅肝：**2-3 次**
  - 猪肉、牛肉：补足剩余餐次（14 顿总量 - 鱼虾鹅肝）

## 规则二：鸡蛋安排

- 每天 **1-2 个**鸡蛋。
- 形式在以下四种间轮换，避免连续重复：
  - 鸡蛋羹
  - 鸡蛋饼 / 小饼（含蛋）
  - 蒸糕（含蛋）
  - 蛋花（入粥/面汤）

## 规则三：主食频次

| 主食类型 | 每周次数 | 说明 |
| --- | --- | --- |
| 粥 | **2 次** | |
| 粗面（碎碎面/疙瘩汤面） | **3 次** | 配鲜虾和蔬菜 |
| 米饭（烩饭） | 其余餐次主力 | 烩饭形式 |
| 蒸糕 / 小饼 / 馒头 | 搭配项，不单独成餐 | 需与其他主食或蛋白并排 |

## 规则四：主食形式要求

- **米饭**：烩饭形式（混合菜、肉汁一起焖煮或拌炒）
- **粗面**：优先配鲜虾和蔬菜
- **蒸糕 / 小饼 / 馒头**：宝宝现阶段量少，需与米饭/粥/粗面组合，不单独构成一顿主食

## 规则五：绿叶菜

- **每顿必须含一种绿叶菜**，计划表中统一写"绿叶菜"占位，品种当天自选。
- 可选品种（已排敏）：西兰花、菠菜、油菜、小白菜、奶白菜、上海青、生菜、油麦菜、菜心
