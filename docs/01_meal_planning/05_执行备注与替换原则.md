# 执行备注与替换原则

## 1. 同类替换原则

> 当某食材当天买不到或宝宝拒绝时，按以下规则等量替换，**不改变当日主蛋白类别和周频次**。

| 原食材 | 可替换为 | 备注 |
| --- | --- | --- |
| 鳕鱼 | 比目鱼 | 同为白肉鱼，互换不影响周频次计数 |
| 比目鱼 | 鳕鱼 | 同上 |
| 黑虎虾 | 北极甜虾 | 同为虾类，互换不影响周频次计数 |
| 北极甜虾 | 黑虎虾 | 同上 |
| 鹅肝 | 猪肝 | 同为肝脏，富铁，互换可行；但猪肝需多漂洗 |
| 碎碎面（粗面） | 疙瘩汤面 | 同为面食，口感稍有区别，可互换 |
| 胚芽米 | 燕麦米 | 同为粗粮主食，互换可行 |
| 蒸糕 | 小饼 | 同为搭配项，均需并入组合餐 |

---

## 2. 烹调原则

- **软烂细碎**：所有食材煮至宝宝可用手指捏碎的软度。
- **少油少盐**：不添加食盐、酱油、糖；油脂优先选核桃油或猪油，用量极少。
- **现做现吃**：尽量不提前一天备好热菜，冷藏后口感变差且营养流失。

---

## 3. 主蛋白轮换逻辑（快速参考）

```
周一：猪肉 → 牛肉
周二：虾   → 鱼
周三：鹅肝 → 牛肉
周四：鱼   → 猪肉
周五：牛肉 → 虾
周六：猪肉 → 鱼
周日：虾   → 鹅肝
```

---

## 4. 未排敏食材引入建议

- 每次仅引入 **1 种**新食材，单独做成小份（不混入已排敏主餐）。
- 观察 **3-5 天**，关注皮疹、便便异常、哭闹异常等反应。
- 通过后在 [02_未排敏食材池.md](./02_未排敏食材池.md) 的排敏记录表中标记，再迁移至 [01_已排敏食材池.md](./01_已排敏食材池.md)。

---

## 5. 新建周计划操作流程

1. 复制 `docs/04_周计划-2026W27.md` 并重命名为对应周次（如 `04_周计划-2026W28.md`）。
2. 对照 [03_排餐规则.md](./03_排餐规则.md) 填写新一周 14 顿菜单。
3. 填写完成后，用"规则校验清单"逐项核对。
4. 执行过程中在"复盘记录表"填写宝宝反馈，指导下周计划调整。
