# 宝宝辅食与健康管理系统

一个专为早产宝宝健康成长研发的家庭管理应用。支持辅食排餐、智能合规校验、疫苗接种倒计时规划、历史儿保生理数据跟踪、25项精细就诊病历归档、以及视力矫正戴眼罩时段累计功能。

---

## 🛠️ 技术栈与架构说明

* **核心技术**：Vue 3 + Vite
* **UI/UX 样式**：Vanilla CSS 原生样式，针对手机（移动端）、平板、PC 进行了全方位的自适应响应式布局适配。
* **数据持久化**：采用纯前端离线持久化技术 —— 浏览器 LocalStorage，即使无网环境也能随时记账。
* **图表库**：Chart.js (用于身高、体重生长曲线趋势分析)。

## 🚀 本地开发与启动指南

### 1. 安装依赖
在项目根目录下执行：
```bash
npm install
```

### 2. 启动本地开发服务 (Hot-Reload)
```bash
npm run dev
```
启动后可在浏览器打开控制台输出的本地端口（如 `http://localhost:5173/`）进行使用。

### 3. 生产环境编译与打包
```bash
npm run build
```
编译完成后会生成 `dist` 静态包目录，可供直接部署上线。
